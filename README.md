# antd-native-slinky

Antd-native 4.0.5 bindings for slinky (powered by ScalablyTyped)

It is distributed for Scala 2.13 and Scala.js 1

```

    "default" %%% "antd-native-slinky" % "0.1.3" 
) 
```
